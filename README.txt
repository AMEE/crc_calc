AMEE Carbon Reduction Commitment Calculator
-------------------------------------------

An example Ruby on Rails application which uses AMEE to calculates emissions and
construct a report based on the UK Government's Carbon Reduction Commitment (see
http://explorer.amee.com/categories/Carbon_Reduction_Commitment for more
details).

gems:
  amee >= 2.0.28
  prawn 0.6.3
  prawn-layout 0.3.2
  hoptoad_notifier
